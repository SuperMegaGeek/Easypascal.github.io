Данный сайт создан для подготовки обучающихся к олимпиадам по информатике.

Сайт содержит банк задач по программированию различной сложности и возможных способов решения. Вам представлено 32 задачи муниципального этапа олимпиады по программированию Нерюнгринского района.

Представленные здесь задачи позволят обучающимся повысить уровень программирования. Для этого достаточно знать различные алгоритмические конструкции и команды языка Pascal.

В каждом разделе вы можете ознакомиться с условиями задач. Тесты представлены в виде входных и выходных данных. Нажав на кнопку «Подсказка» вы перейдёте на страницу, на которой расписан способ решения задачи. Нажав на кнопку «Решение» начнётся загрузка файла решения .pas. Для открытия файла с расширением .pas требуется скачать среду разработки PascalABC.net перейдя по ссылке.
